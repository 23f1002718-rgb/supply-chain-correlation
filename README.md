# Supply Chain Correlation Analysis

This repository contains the correlation matrix and heatmap generated from the supply chain dataset (62 procurement transactions).

- `correlation.csv`: Correlation matrix between Supplier_Lead_Time, Inventory_Levels, Order_Frequency, Delivery_Performance, and Cost_Per_Unit.
- `heatmap.png`: Excel heatmap using Red–White–Green conditional formatting.
